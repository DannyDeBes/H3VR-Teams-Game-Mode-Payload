# Sosig Team

## Sosig Team
| Variable    | Setting |
| -------- | ------- |
| name  | Sosig Team name |
| description | A Brief description of the sosig team |
| sosigSet | A collection of Sosig Sets that will be randomly selected from when spawning a sosig |

### Sosig Set
| Variable    | Setting |
| -------- | ------- |
| minKills  | The minimum amount of kills required before this sosig set begins spawning |
| maxKills  | The maximum amount of kills before it stops spawning |
| sosigEnemyIDs  | A list of all Sosig Enemy IDs that will be randomly selected from when this set is chosen |
