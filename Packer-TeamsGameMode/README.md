# Teams Game Mode Plugin
Teams Game Mode (TGM) is a system for handling team based gamemodes all under one roof, this system allows mappers to create 1 map that automatically supports any valid game modes. TGM supports player classes, sosig teams, sosig count per team, profile saving and load, and more!

TGM is also fully mod supported for custom maps, sosig teams and player teams!

### Current Supported Gamemodes:
* Team Deathmatch
* Rush
